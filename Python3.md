# 第3章 Python的基本类型

## 3-2 数字：整型与浮点型

整数：int

浮点数：float

```type()```：可以判断类型

* ```type(1 * 1)```：```<class 'int'>```
* ```type(1 * 1.0)```：```<class 'float'>```

* ```type(2 / 2)```：```<class 'float'>```

* ```type(2 // 2)```：```<class 'int'>```

  ```2 / 2```：```1.0```

  ```1 // 2```：```0```

## 3-4 各进制的表示与转换

表示：

* 二进制
  * ```0b10```：2
* 八进制
  * ```0o11```：9
* 十六进制
  * ```0x1F```：31

转换：

* 二进制
  * ```bin(10)```：```'0b1010'```
  * ```bin(0xE)```：```'0b1110'```
* 十进制
  * ```int(0b111)```：```7```
* 八进制
  * ```hex(0o7777)```：```'0xfff'```
* 十六进制
  * ```oct(0b111)```：```'0o7'```
